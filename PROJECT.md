## 📌 プロジェクト概要

**funfo AI Store** は、飲食店オーナー向けの **AI アプリ生成プラットフォーム**です。

コードを書かずに、自然言語（日本語）でアプリを作れるというコンセプト。Bolt.new / Base44 / v0.dev のような「Vibe Coding」スタイルを、飲食店業務（売上管理・シフト管理・在庫管理など）に特化させたSaaSプロダクトです。

funfo POSシステムのデータ（CSV）と連携し、飲食店の経営データをリアルタイムで可視化・分析できるアプリをユーザー自身が生成・カスタマイズできます。

## 📱 画面仕様

### 1. ログイン画面（`/login`）

**目的:** ユーザー認証のエントリーポイント

**機能:**
- 単店舗 / 多店舗の2モードをタブで切り替え
- アカウント・パスワード入力フォーム
- 「パスワードを忘れた方」→ `/forgot-password` へ遷移
- ログイン成功 → `/`（メイン画面）へ遷移
- デモ用: 空のまま送信でもログイン可能

**実装状態:** ✅ UIプロトタイプ完成（認証ロジックは未実装）

---

### 2. パスワード回復画面（`/forgot-password`）

**目的:** パスワードをメール認証でリセット

**フロー（2ステップ）:**
1. **Step 1:** メールアドレス入力 → 「認証コードを送信」
2. **Step 2:** 6桁認証コード + 新パスワード + 確認 → 「パスワードをリセット」

**機能:**
- 60秒カウントダウン付き再送信ボタン
- パスワード不一致チェック
- ログインに戻るボタン

**実装状態:** ✅ UIプロトタイプ完成（メール送信・API連携は未実装）

#### タブ 2: ワークスペース（`workspace`）

**目的:** AIチャットでアプリを生成・編集するエディタ

**2つのモード:**

**① リストモード（デフォルト）**
- マイアプリをカードグリッドで表示（3列）
- 各カード: アイコン / 名前 / バージョン / 更新日時 / ステータス（公開中/下書き）
- 「編集」→ エディタモードへ遷移
- 「新規アプリ作成」ボタン

**② エディタモード（AI チャット + プレビュー）**
- **バージョン履歴バー（上部）:** 最大4バージョン表示、最新にバッジ
- **左ペイン（AI チャット）:**
  - AIメッセージ / ユーザーメッセージの会話UI
  - 生成中ステータスカード（チェックリスト + スピナー）
  - コードプレビューカード（シンタックスハイライト）
  - 入力エリア: ファイル添付 + テキストエリア + 送信ボタン
- **リサイズハンドル:** ドラッグで左右ペインの幅調整
- **右ペイン（プレビュー）:**
  - デバイス切り替え: 📱モバイル / 💻ラップトップ / 🖥デスクトップ
  - URLバー表示
  - モックダッシュボード（売上報告のサンプル）

**プレビューで表示されるモックデータ:**
- 今日の売上: ¥84,320（↑12.4%）
- 注文数: 1,247（↑8.1%）
- 客単価: ¥67.6（↑3.9%）
- 今月累計: ¥1.8M（達成率74%）
- 過去7日棒グラフ
- 料理別売上ランキング（3品）

**実装状態:** ✅ UIプロトタイプ完成（AI生成・コード実行は未実装）

## 📊 サンプルデータ（funfo POS連携）

`src/imports/` に funfo POSシステムのエクスポートCSVサンプルが格納されています。

| ファイル | 内容 |
|---------|------|
| `funfo_Demo店_商品別_2026年02月01日_~_2026年02月28日.csv` | 商品別売上（商品ID/名/カテゴリー/単価/販売数/売上高/原価/粗利）|
| `funfo_Demo店_商品別_...-1.csv` | 同上（別バージョン）|
| `funfo_Demo店_取引詳細_2026年02月01日_~_2026年02月28日.csv` | 取引詳細（レシートNo/日時/合計/支払方法）|
| `funfo_Demo店_取引詳細_...-1.csv` / `-2.csv` | 同上（別バージョン）|
| `funfo_Demo店_オプション別_2026年02月01日_~_2026年02月28日.csv` | オプション別売上 |
| `funfo_Demo店_カテゴリー別_2026年02月01日_~_2026年02月28日.csv` | カテゴリー別売上 |
| `funfo_Demo店_部門別_2026年02月01日_~_2026年02月28日.csv` | 部門別売上 |
| `funfo_Demo店_支払方法別売上_2026年03月.csv` | 支払方法別売上（現金/PayPay/Rakuten/Square等）|
| `店舗別売上統計_2026年03月03日_~_2026年03月03日.csv` | 多店舗売上統計 |
| `グランブルール&京都直_売上詳細_2026年.csv` | 複数店舗の売上詳細 |

`src/data/receiptData.ts` にはレシートデータ80件がTypeScriptオブジェクトとして用意されています。

**レシートデータの主要フィールド:**
```
date, receiptNo, type（通常/訂正/赤伝票）, symbol（テーブル番号）
checkinTime, checkoutTime, subtotal, discount, total
customers, orders
cash, paypay, rakuten, square, squareTerminal, voucher, funfo, urikake, point
```
